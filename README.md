Aplikasi Produk Sederhana (Flutter)
Aplikasi Flutter sederhana untuk menampilkan produk dengan fitur penambahan dan pengurangan jumlah (quantity), serta menghitung total harga secara otomatis.

📱 Tampilan Aplikasi
Aplikasi ini menampilkan:
1. Nama produk
2. Gambar produk
3. Harga produk
4. Total harga
5. Tombol tambah (+) dan kurang (-)
   
✨ Fitur Utama
1. Menampilkan data produk (nama, gambar, harga)
2. Menambah dan mengurangi jumlah produk
3. Menghitung total harga secara otomatis
4. Tampilan UI sederhana menggunakan Flutter
   
Cara Kerja
Variabel qyt digunakan untuk menyimpan jumlah produk, Variabel price menyimpan harga satuan, dan Variabel total dihitung dengan rumus: total = price * qyt

Setiap tombol ditekan:
Tombol ➕ menambah jumlah, 
Tombol ➖ mengurangi jumlah (tidak bisa minus),
dan UI diperbarui menggunakan setState()
